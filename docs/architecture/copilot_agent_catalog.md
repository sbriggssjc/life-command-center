# Copilot Agent Catalog

## Purpose
Document the initial and future Copilot agents, their responsibilities, boundaries, inputs, outputs, and operational surfaces.

## Outline
1. Agent naming and taxonomy
2. Core agent definitions (Daily Briefing, Intake & Triage, Prospecting, Listing Pursuit, Marketing & Seller Reporting, Deal Execution)
3. Ownership boundaries and system-of-record constraints
4. Trigger patterns and invocation contexts
5. Tool/action access policy by agent
6. Human-in-the-loop and approval rules
7. Telemetry and quality evaluation per agent
8. Expansion roadmap by wave
